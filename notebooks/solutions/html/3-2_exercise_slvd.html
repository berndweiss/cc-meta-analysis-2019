Exercise¶

The next step is to run a so-called Eggers regression test, which was covered in the section on Correlational and regression approaches.

Despite the fact that the metafor package provides a ready-to-use function called regtest(), we will start by using a simple regression model. In R, the linear model can be estimated using the lm(y ~ x, data = ...) function (see also the Monday exercises).

Tip: You might want to recall that you have to convert the effect sizes to the so-called standard normal deviate (SND). The precision is defined as 1/SE i
